# Checkers


Gra w warcaby napisana w języku javaScript przy pomocy biblioteki jQuery.

Funkcjonalności :

-Tworzenie planszy,
-Liczenie punktów przy zbiciu,
-Poruszanie,
-Zbijanie do przodu,
-Podział na tury

Przyszłe fukcjonalności :

-Bicie przymusowe,
-Zbijanie do tyłu,
-Zbijanie podwójne/potrójne itd,
-Damka,
-Podział na moduły.
